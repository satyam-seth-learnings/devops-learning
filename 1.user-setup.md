# User Setup

- Add new user in linux

    ```bash
    su adduser <try-nginx>
    ```

- Check linux users

    ```bash
    cut -d: -f1 /etc/passwd
    ```

- Check ipaddress of linux server

    ```bash
    Hostname -I
    ```

- Change user

    ```bash
    su <username>
    ```

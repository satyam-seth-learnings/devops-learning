# Install Node and NPM

- Check if Node already Installed

    ```sh
    node -v
    ```

- Check if NPM already Installed

    ```sh
    npm -v
    ```

- Install Node and NPM

    ```sh
    curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - && \
    sudo apt-get install -y nodejs
    ```
